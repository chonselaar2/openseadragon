OpenSeadragon example: legacy image pyramids

A "legacy image pyramid" just means we can finally make use of our existing images. Web developers have commonly created derivatives of various sizes that are intended for use for different web presentations or for download, like "thumbnail", "detail", and "best" etc. This is already a simple pyramid and you can use them to create a simple deep zoom presenation with OpenSeadragon!

Try it on the iPad! (new since 0.9)
