OpenSeadragon example: binding custom butoms

You can now bind to an entirely custom user interface by just passing the id of the elements you want treated as buttons! This is especially useful when you need to fully control the look and feel of the user interface. (* new since 0.9.50) See the "Binding custom buttons" example below.
